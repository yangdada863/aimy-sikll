# settings.json 

保留Hooks

```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "<填写你的 API Key>",
    "ANTHROPIC_BASE_URL": "<填写你的 API 地址>",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "<填写 Haiku 模型 ID>",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "<填写 Sonnet 模型 ID>",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "<填写 Opus 模型 ID>"
  },
  "permissions": {
    "defaultMode": "bypassPermissions"
  },
  "hooks": {
    "UserPromptSubmit": [
      {
        "hooks": [
          {
            "type": "command",
            "command": "python C:/Users/西山/.claude/hooks/security-context-hook.py"
          }
        ]
      }
    ]
  },
  "autoMemoryEnabled": true,
  "skipDangerousModePermissionPrompt": true
}
```

## 字段说明

| 字段 | 说明 |
|------|------|
| `ANTHROPIC_AUTH_TOKEN` | API 认证密钥 |
| `ANTHROPIC_BASE_URL` | API 基础地址（如使用代理需填写） |
| `ANTHROPIC_DEFAULT_HAIKU_MODEL` | Hook LLM 分类器使用的模型 |
| `ANTHROPIC_DEFAULT_SONNET_MODEL` | 默认 Sonnet 模型 |
| `ANTHROPIC_DEFAULT_OPUS_MODEL` | 默认 Opus 模型 |
| `hooks.UserPromptSubmit` | **核心配置** — 注册安全研究上下文注入 Hook |
| `permissions.defaultMode` | 权限模式，`bypassPermissions` 跳过逐次确认 |
| `autoMemoryEnabled` | 启用自动记忆 |
| `skipDangerousModePermissionPrompt` | 跳过危险模式提示 |
